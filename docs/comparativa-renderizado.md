# Comparativa de Técnicas de Renderizado Web

## CSR (Client-Side Rendering)
- Ventajas: Interactividad inmediata, menor carga inicial.
- Desventajas: Dependencia del cliente, posible retraso en la carga.

## SSR (Server-Side Rendering)
- Ventajas: Mejor SEO, carga inicial más rápida.
- Desventajas: Mayor complejidad, dependencia del servidor.

## Pre-render
- Ventajas: Mejor SEO, carga inicial muy rápida.
- Desventajas: Dificultad en la actualización dinámica.
